### Rodando a Calculadora de imc
~~~bash
# Clone este repositório
$ git clone <https://github.com/Demanuel001/imc-reactnative.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd imc-reactnative

# Execute a aplicação em modo de desenvolvimento
$ expo start

~~~
